Projects

Have completed various projects both during university and in my own free time, most of which are open source and hosted on my GitHub account.

  • Statistical analysis tool - Extended functionality of a network forensics tool, written by the university Centre for Software Reliability (CSR team), using Java:
    1. Created 16 automated unit tests with JUnit (where before there were none).
    2. Added code for analysis of large batches of data at once, saving 100s of clicks per run.
    3. Refactored existing code to use more efficient data structures for storing parsed data.

  • Mock air ticket sales system (not open source/hosted online) - Created a piece of software from scratch, in a team of 4 people, for organising air ticket sales:
    1. Analysed requirements and designed the entire system using UML.
    2. Used Java Swing library, to create a GUI for the frontend of the system.
    3. Used SQL with Java to create a backend to handle user accounts for Travel Advisors, Office Managers and System Administrators.

  • Visalisation of London air quality - Created a data visualisation that showed data regarding amounts of greenhouse gases in London using Processing (similar to D3.js):
    1. Data set spanned 10 years (2008-2017) and contained almost 3000 records of data.
    2. Lightweight, optimised to run with a <0.1s response time to all user input.
    3. Efficient colour encoding; used to make differentiating between data easier on eyes.

  • Python command-line interface games - Two cross-platform games made to be played in the terminal:
    1. Tic-tac-toe - can be played with two players locally.
    2. Blackjack - play against the dealer with a full betting system.
